Textdraw is a text drawing library for Dyalog APL.

- Basic drawing functions: boxes, pattern filling, text, bitmaps
- Plotting: scatter, line and area plots, bars, histograms, heat maps
- Tables: with text and numeric columns, inline plots, and level bars

A fixed width font with the used glyphs is required.
[APL387](https://dyalog.github.io/APL387/) is recommended.
